# pin_pong
---
Пинпонг- игра в которой игрокам необходимо отбивать мяч используя ракетки.

Если противник пропустил мяч вы получаетее 1 балл.

В сете(раунде) побеждает игрок который первым набрал определённок кол-во баллов.

В партии побеждает игрок который первым выйграл определённое кол-во сетов.

### Запуск пректа
---
Убедитесь что скачали и загрузили все необходимые файлы из репозитория файлы из репозитория.

В противном случае программа не запустится.

Для запуска проекта в терминале необходимо ввести команду:
```
python pin_pong_game.py
```


### Создание исполняемого файла
---
Для того чтобы вам не приходилось постоянно заходить в редактор кода

и запускать проект через терминал его нужно превратить в исполнительный.

***Для Windows***

1 Убедитесь что у вас установлен pyinstaller. Для этого в терминал введите:
---
  ```
  pip list
  ``` 
  Если его нет то скачайте его командой:
  ```
  pip install pyinstaller
  ```
 2 Замените все шрифты на стандартные.
 ---
  Было:
  ```
  font = font.Font(none,50)
  ```
  Стало:
  ```
  font = font.SysFont('Arial',50)
  ```
  Чтобы проще найти шрифты используйте F3 и в окно введите Font
  
3 Убедитесь что прект работает.
---
  Зате в терминале введите команду 
  ```
  pyinstaller --onefile pin_pong_game.py
  ```
4 Создайте новую папку и скопируйте туда:
---

  Исполнительный файл(.exe) из папки dist
  
  Все графические и звуковые файлы
